# Skills locais

Coloque aqui as skills que deseja manter junto do projeto NexoDoc.

Estrutura sugerida:

```text
.codex/skills/
└─ nome-da-skill/
   └─ SKILL.md
```

Depois de adicionar uma skill, peça ao Codex:

```text
Leia .codex/skills/nome-da-skill/SKILL.md e use como referencia nesta tarefa.
```
