# Template Prefeitura Municipal de Sao Jose

Arquivo ODT oficial do template de capa da Prefeitura Municipal de Sao Jose.

## Caracteristicas

- Volume em formato romano (Vol. I, Vol. II...)
- Orgao e secretaria estao fixos no design do template
- 7 marcadores: NOME_OBRA, FASE, TITULO_CAPA, TOMO, VOLUME, MES_ANO, CODIGO_EXIBIDO
